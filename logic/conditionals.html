A conditional statement (also called an if-then statement) links two statements together with a cause-and-effect relationship.

Form: If p, then q ‚Äî written as p ‚Üí q

  • p is called the hypothesis (the condition / the "if" part)

Examples

p = "It rains"  |  q = "The ground gets wet"

If a number is divisible by 4, then it is even.
Always true ‚Äî this is a mathematical conditional.

üîÉ The Four Related Conditionals

From any conditional p ‚Üí q, we can form three related statements by swapping or negating:

Original (Conditional): p ‚Üí q

"If it is a dog, then it is a mammal."

Converse: q ‚Üí p  (swap p and q)

"If it is a mammal, then it is a dog."  ‚Äî FALSE (cats are mammals too!)

The converse is not necessarily equivalent to the original.

Inverse: ¬¨p ‚Üí ¬¨q  (negate both)

"If it is not a dog, then it is not a mammal."  ‚Äî FALSE

The inverse is not necessarily equivalent to the original either.

Contrapositive: ¬¨q ‚Üí ¬¨p  (swap AND negate)

"If it is not a mammal, then it is not a dog."  ‚Äî TRUE

The contrapositive is always logically equivalent to the original. If the original is true, the contrapositive is true ‚Äî and vice versa.
