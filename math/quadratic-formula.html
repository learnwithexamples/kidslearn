The Quadratic Formula

Any quadratic equation written as ax² + bx + c = 0 (where a ≠ 0) can be solved with this formula:

x = −b ± √(b² − 4ac) 2a

The Coefficients

a = number in front of x² b = number in front of x c = the constant term

The Discriminant

Δ = b² − 4ac

The part under the square root tells you how many real solutions there are.

Δ > 0

Two real solutions

The ± gives two different answers.

Δ = 0

One repeated solution

x = −b / 2a

Δ < 0

No real solutions

The square root of a negative number is not a real number.

Vertex (turning point)

x = −b / 2a

The lowest or highest point of the parabola.

Worked Example

Solve x² − 5x + 6 = 0

1. Identify the coefficients: a = 1, b = −5, c = 6
2. Discriminant: Δ = (−5)² − 4(1)(6) = 25 − 24 = 1
3. Square root: √1 = 1
4. Apply the formula: x = (5 ± 1) / 2
5. Two solutions: x = 6/2 = 3 and x = 4/2 = 2

Tips

  • Watch the signs! A negative b becomes positive when you compute −b.
  • Always rearrange the equation so one side equals 0 first.
  • If b² − 4ac is a perfect square, the answers will be nice whole numbers or fractions.
  • You can check your answers by substituting them back into the original equation.
